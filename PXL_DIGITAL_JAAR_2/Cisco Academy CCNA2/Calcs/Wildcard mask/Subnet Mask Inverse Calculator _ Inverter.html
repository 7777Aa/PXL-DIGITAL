Subnet Mask Inverter

Subnet Mask Inverse Calculator is a small piece of online network tool to invert the dotted decimal subnet mask IP addresses into inverted Subnet Mask IP. The inverse mask of a Subnet Mask IP Address can be easily calculated by subtracting each IP value from 255. This inverse subnet mask is generally used in ACLs. When you try yourself solving such calculations, this Subnet Mask Inverse Calculator can be used to verify your results.
