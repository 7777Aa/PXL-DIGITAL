#Configuring Dynamic NAT
ip nat pool SPECIFY NAME START-IP END-IP {netmask | prefix-length}
access-list SPECIFY ACCES-LIST-NUMBER permit SPECIFY SOURCE SOURCE-WILDCARD
ip nat inside source list SPECIFY ACCESS-LIST-NUMBER pool SPECIFY NAME
interface SPECIFY INTERFACE
ip nat inside
interface SPECIFY INTERFACE
ip nat outside
