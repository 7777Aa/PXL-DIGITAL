

OSPF

een proces id meegeven:
Router (config) # router ospf 1 
router-id 1.1.1.1 // router een id geven 
network 192.168.1.0 0.0.0.255 area 0 // area maken en toevoegen aan area
network 192.168.5.0 0.0.0.3 area 0
network 192.168.5.12 0.0.0.3 area 0 
passive-interface g0/0 

show ip ospf ? // geeft de opties voor info over OSPF

show ip ospf database // laat zien wat allemaal uitgewisseld wordt

show ip ospf neighbours // laat de buren zien 

++ bandbreedte consistent maken ++
auto-cost reference-bandwidth 10000 // zorgt ervoor dat alle bandbreedte consistent is, dit moet dan wel op elke router ingesteld worden best als ge dit al doet op een router 

int s0/0/0 
bandwidth 64
exit

als ge veranderingen doet, uw OSPF proces restarten!
